01-flight-basic 01-flight-rounded 01-flight-vtol 飞行

上海陆家嘴 3D 飞行系列。基础圆形航线、圆角四边形航线（转弯倾斜 + ENU 近距跟随）、VTOL 模型替换，共享 SceneManager / RoundedFlightPath / FlightTracker 模块。2026.05

02-airspace 空域

虹桥(双跑道)与浦东(五跑道)机场 B 类空域 3D 可视化，依据《国家空域基础分类方法》分层展示，玻璃拟态控制面板 + 底部总览按钮。2026.05

03-particles 特效

双飞机航线粒子效果，支持火焰 / 烟雾 / 气流三种特效切换与强度调节，粒子物理模拟（浮力、湍流）。2026.05

04-weather 04-weather-cloud 天气

雷雨云天气飞行演示（云团、细雨、闪电）+ drizzle / rainstorm / darkStorm 预设切换；04-weather-cloud 独立加载 rain_1.glb 云模型。2026.05
